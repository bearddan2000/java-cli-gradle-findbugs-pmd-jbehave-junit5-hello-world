# java-cli-gradle-findbugs-pmd-jbehave-junit5-hello-world

## Description
A POC app using junit5,
pmd, jbehave framework.

## Tech stack
- java
- gradle
  - junit5
  - jbehave
  - pmd
  - findbugs

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
